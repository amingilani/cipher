Canadian Index of Practical Ham Encryption Resources / Index Canadien des Ressources de Chiffrement pour Radioamateur


About / À propos
--------------

Welcome to CIPHER (Canadian Index of Practical Ham Encryption Resources), an open-source, community-maintained documentation suite for encryption protocols used in Canadian amateur radio communications.

Bienvenue au CIPHER (Index Canadien des Ressources de Chiffrement pour Radioamateur), une suite de documentation open-source, maintenue par la communauté, pour les protocoles de chiffrement utilisés dans les communications radio amateur canadiennes.

Purpose
-------

This repository documents encryption protocols for amateur radio, compliant with Canadian regulations which permit encrypted communications provided the protocol and keys are publicly documented. As per Radio Amateurs of Canada, who maintain a `public registry of encryption keys <https://rac.ca/encryption>`_, any operator may use encryption as long as the method remains transparent and documented.

Ce dépôt documente les protocoles de chiffrement pour la radio amateur, conformes aux règlements canadiens qui permettent les communications chiffrées à condition que le protocole et les clés soient documentés publiquement. Selon Radio Amateurs du Canada, qui maintient un `registre public des clés de chiffrement <https://rac.ca/encryption>`_, tout opérateur peut utiliser le chiffrement tant que la méthode reste transparente et documentée.
